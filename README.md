# Healthcare-benefits-enrollment-dashboard
Power BI dashboard analyzing healthcare benefits enrollment, dependents, and employee coverage trends.

 Healthcare Benefits Enrollment Dashboard

## Project Overview
This Power BI dashboard analyzes healthcare benefits enrollment data and provides insights into employee enrollment trends, dependent coverage, and plan participation.

## Dashboard Pages

### Executive Summary
- Total Employees
- Active Members
- Terminated Members
- Enrollment by Department
- Enrollment by Plan Type
- Enrollment Trend
- Enrollment by Location

### Dependent Analysis
- Total Dependents
- Children and Spouse Distribution
- Dependents per Employee
- Dependents by Age Group

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key Insights
- Employee enrollment trends across locations and departments
- Dependent demographics and coverage analysis
- Healthcare plan participation patterns
